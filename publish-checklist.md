# Dibs — Publish Checklist

Remaining manual steps before submitting to the Figma Community marketplace.

## Figma Developer Console

- [ ] Go to figma.com/developers and register Dibs as a new plugin
- [ ] Copy the generated numeric plugin ID into `manifest.json` (replace `"garden-planner-sfg"`)
- [ ] Join the Figma Community creator program: Profile → Community → "Become a creator"

## Listing Assets

- [ ] Create a cover image (1920×960px) and upload it in the developer console — this is the biggest discoverability driver

## Testing

- [ ] Test the plugin on a shared or team file (not just personal drafts) to confirm permissions behave correctly

## Submit

- [ ] Fill out the listing in the developer console (name, descriptions, tags, cover art — see `store-listing.md`)
- [ ] Submit for review — Figma's review typically takes a few business days
